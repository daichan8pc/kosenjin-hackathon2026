# 高専人恋愛デバッガー「Love_PostMortem.exe」

![License](https://img.shields.io/badge/license-MIT-blue)
![Version](https://img.shields.io/badge/version-1.0.0-green)
![Team](https://img.shields.io/badge/Team-404-red)

**～なぜ、我々の恋愛はビルドエラーを起こすのか～**

高専生による、高専生のための、恋愛力診断アプリケーションです。
150問以上のデータベースからランダムに出題される問いに答え、あなたの「恋愛エンジニアリング力」を測定します。

## デモサイト (Demo)
[こちらから試験を受ける](https://404lovenotfound.vercel.app/)
## 特徴 (Features)

* **高専生特化**: 全国の国公私立高専を選択可能。高専生特有の文脈を考慮した設問設計。
* **リアルな判定**: 正答率に基づき S〜D の5段階でランク付け。
* **免許証発行**: 合格者（Cランク以上）には、「恋愛免許証」を発行（現在未実装）。
    * ※ Dランク（不合格）の場合は発行不可（落単扱い）
* **レスポンシブデザイン**: スマホ、PCどちらでも美しい「Team 404」カラーのUI。

## 使用技術 (Tech Stack)

* **Frontend**: HTML5, CSS3, Vanilla JavaScript
* **Font**: [Zen Kurenaido](https://fonts.google.com/specimen/Zen+Kurenaido) (Google Fonts)
* **Deployment**: Vercel (Static Site Hosting)
